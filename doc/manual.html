Section 01

产品简介

Orchestra 是 Mafdet.AI 出品的 AI 多角色协同工作流平台。就像一支管弦乐团，Orchestra 让多个 AI 角色各司其职、协同配合，共同完成复杂的分析与研究任务。

工作流主页 — 清晰展示可用的工作流列表

核心工作原理

每次执行工作流，系统会按照以下流程协同运作：

1

🎯 引导者（Conductor）分析任务

首先由引导者 AI 深入理解你的任务描述，将复杂问题拆解为多个专项子任务，并为每位专家分配明确的研究方向。

2

🔬 多位专家并行处理

多位专家 AI 同时并行工作，每位专家专注于自己的领域（如研究、风险、策略等），互不干扰，大幅缩短总耗时。

3

📊 汇总者整合输出

所有专家完成后，汇总者 AI 将各方观点融合整理，生成一份结构完整、逻辑清晰的综合报告，可直接下载为 PDF。

Section 02

快速上手

首次使用 Orchestra，按照以下 4 个步骤即可完成第一次工作流执行：

1

登录账号

访问 Orchestra 网址，点击登录按钮，通过 OAuth 授权完成登录。登录后系统会记住你的会话状态。

2

添加 AI 模型

点击底部导航栏的「模型管理」，添加你的 AI 服务 API Key（如 DeepSeek、OpenAI 等）。这是使用专家功能的前提。

3

（可选）配置专家角色

进入「流程配置」，为每位专家选择已添加的模型，并根据需要调整系统提示词。如果不需要自定义，使用默认配置即可。

4

执行工作流

进入「执行」页面，在输入框中描述你的任务，点击「开始执行」。系统将自动完成引导→专家→汇总的全流程。

Section 08

常见问题

专家的 AI 模型下拉列表是空的，怎么办？
这是因为还没有在「模型管理」页面添加任何模型配置。请先点击底部导航栏的「模型管理」，添加至少一个 AI 服务的 API Key，之后下拉列表就会显示可选项。
任务执行失败，提示 API 错误，如何排查？
常见原因：
  • API Key 无效或过期：前往模型管理页删除旧配置，重新添加有效的 API Key
  • 账户余额不足：登录对应服务商平台充值
  • 模型名称错误：确认模型标识符与服务商文档一致（如 deepseek-chat 而非 deepseek）
  • 网络问题：部分服务商 API 需要特定网络环境才能访问
刷新页面后历史任务还在吗？
是的，所有执行记录都保存在数据库中，刷新页面、关闭浏览器、重新登录后都可以在「历史记录」页面找到之前的所有任务和报告。
可以同时执行多个工作流吗？
可以。每次点击「开始执行」会创建一个独立的任务，多个任务可以同时在后台运行。在「任务状态」页面可以看到所有正在进行的任务。
如何获取 DeepSeek API Key？
  1. 访问 platform.deepseek.com 注册账号
  2. 进入「API Keys」页面，点击「创建 API Key」
  3. 复制生成的 Key（以 sk- 开头）
  4. 在 Orchestra 的模型管理页面，选择 DeepSeek 服务商，粘贴 API Key，模型名填写 deepseek-chat
注册后通常会赠送一定的免费额度，足够体验使用。
