# Orchestra (Open Source Shell)

Orchestra 是 Mafdet.AI 的多角色 AI 工作流框架开源空壳版本，不包含任何预置工作流或测试业务数据。

![Hero Section（首页头图）—— 指挥官的诞生](./doc/hero-commander-birth.jpg)

## 中文操作说明

1. 环境要求：Node.js 20+、pnpm 9+、PostgreSQL。
2. 安装依赖：`pnpm install`
3. 配置环境变量：复制 `.env.example` 为 `.env`，并填写数据库与 API 配置。
4. 初始化数据库：`pnpm db:push`
5. 启动开发环境：`pnpm dev`
6. 前端（Web/Metro）默认地址：`http://localhost:8081`
7. 后端 API 默认地址：`http://localhost:3000`

如需要体验，请访问 [www.mafdet.ai](https://www.mafdet.ai)。

## English Quick Start

1. Requirements: Node.js 20+, pnpm 9+, PostgreSQL.
2. Install dependencies: `pnpm install`
3. Configure env: copy `.env.example` to `.env`, then fill in DB and API settings.
4. Initialize database: `pnpm db:push`
5. Start dev mode: `pnpm dev`
6. Frontend (Web/Metro) default: `http://localhost:8081`
7. Backend API default: `http://localhost:3000`

For product experience, visit [www.mafdet.ai](https://www.mafdet.ai).
